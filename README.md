# 🚀 Aavash Regmi - Premium Personal Portfolio

A world-class, full-stack premium portfolio website built with modern technologies and futuristic design.

## 🎨 Features

- ✨ Ultra-premium dark theme with golden accents
- 🎭 3D animations and visual effects
- 🔐 Professional authentication system with secure password reset
- 💌 Contact form with email integration
- 📊 Real-time age counter

- 📱 Fully responsive design
- ⚡ Smooth transitions and interactions

## 🛠 Tech Stack

**Frontend:**
- React + Vite
- Tailwind CSS
- Framer Motion
- Three.js (3D effects)
- Axios

**Backend:**
- Node.js + Express
- MongoDB
- JWT Authentication
- Nodemailer

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB
- Gmail account with App Password

### Installation

1. Clone the repository
```bash
git clone https://github.com/aavashregmi/aavashregmi.github.io.git
cd aavashregmi.github.io
```

2. Install backend dependencies
```bash
npm install
```

3. Set up environment variables
```bash
cp .env.example .env
# Edit .env with your credentials
```

4. Install frontend dependencies
```bash
cd client
npm install
cd ..
```

5. Run development server
```bash
npm run dev
```

The application will be available at `http://localhost:5173` (frontend) and `http://localhost:5000` (backend)

## 📧 Email Configuration

1. Enable 2FA on your Gmail account
2. Create an App Password
3. Add to `.env`:
   ```
   EMAIL_USER=your_email@gmail.com
   EMAIL_PASS=your_app_password
   ```
